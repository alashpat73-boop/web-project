# Web Project - Task Manager

وصف المشروع: مشروع بسيط لإدارة المهام (task manger).

## نظرة عامة
هذا المستودع يحتوي على تطبيق ويب لإدارة المهام مبني باستخدام JavaScript، PHP، HTML، وCSS. يوجد أيضًا ملفات Dockerfile وأدوات PowerShell بسيطة.

## المميزات
- إنشاء، تعديل، وحذف المهام
- واجهة مستخدم ويب بسيطة
- قد يحتوي على إعدادات للنشر باستخدام Docker

## البنية التقنية
اللغات الأساسية في المشروع:
- JavaScript
- CSS
- PHP
- HTML
- Dockerfile
- PowerShell

## المتطلبات
- PHP >= 7.4
- خادم ويب (Apache أو Nginx) أو Docker
- Node.js (إذا كان هناك أدوات تطوير تعتمد على npm)

## التثبيت (محلي)
1. استنساخ المستودع:
   git clone https://github.com/alashpat73-boop/web-project.git
2. ادخل إلى المجلد:
   cd web-project
3. إعداد بيئة PHP وملف التهيئة (إن وُجد):
   - ضع إعدادات قاعدة البيانات في ملف الإعدادات (راجع الملفات الموجودة)
4. شغل الخادم المحلي أو افتح الملفات عبر خادم الويب المحلي.

## تشغيل باستخدام Docker (إن وجد Dockerfile)
1. بناء الصورة:
   docker build -t web-project .
2. تشغيل الحاوية:
   docker run -p 8080:80 web-project
3. افتح المتصفح على http://localhost:8080

## بنية المجلد (توقعية)
- /public أو /www — ملفات الواجهة (HTML/CSS/JS)
- /src أو /app — منطق الخادم (PHP)
- /docker — ملفات إعداد Docker (إن وُجدت)
